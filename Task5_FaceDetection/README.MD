# 📌 Project Overview

This project is an **AI-powered Face Detection System** developed during the **CODSOFT AI Internship 2026**.

It uses **OpenCV Haar Cascade Classifiers** — a machine learning based approach where a cascade function is trained on thousands of positive and negative images to detect faces and eyes in real time.

---

# ✨ Features

👁️ Detects faces in any uploaded image
🔵 Detects eyes with blue bounding boxes
🟢 Draws green bounding boxes around faces
📊 Shows total face & eye count
🎨 Beautiful animated dark UI
⚡ Fast real-time detection
📱 Works with any image format
🤖 Powered by pre-trained Haar Cascade model

---

# 🧠 How It Works

```yaml
Step 1: User uploads any image
Step 2: Image converted to grayscale
Step 3: Haar Cascade scans image
Step 4: Detects face regions
Step 5: Detects eyes within face regions
Step 6: Draws bounding boxes
Step 7: Returns annotated result
```

# 🧠 About Haar Cascade

Haar Cascade is a **machine learning based object detection algorithm** proposed by Paul Viola and Michael Jones in 2001.

- Trained on **thousands of face images**
- Uses **sliding window** technique
- **Cascade of classifiers** for fast detection
- Industry standard for real-time face detection
- Used in cameras, security systems & apps
